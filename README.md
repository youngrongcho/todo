# My Todo App

Todo 관리를 위한 웹 애플리케이션입니다.

## Features

- Todo를 쉽게 생성하고 삭제할 수 있습니다.
- Todo에 우선순위를 설정할 수 있습니다.
- Spring Boot로 쉽게 서버 배포를 할 수 있습니다.
